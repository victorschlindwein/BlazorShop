# BlazorShop

Esse projeto foi criado com intuito de explorar a nova atualização do Blazor lançada pela Microsoft.

É um projeto muito simples trata-se de um CRUD de produtos e categorias.

Ele foi construído em conjunto com o Balta durante o curso de [Fundamentos do Blazor com .NET 8.](https://balta.io/player/assistir/6263333a-258b-4077-9627-96e575e48125/6e1069ee-2434-4eee-9c30-d6e3fcfd33ad)
